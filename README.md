# 📊 SharesApp – Indian Stock Research & Analysis Platform

SharesApp is a full-stack web application built to help users **analyze and research Indian stocks** in a simple and powerful way. It provides complete insights including fundamentals, news, sentiment analysis, and AI-based recommendations.

> ⚠️ This application is for **educational and research purposes only**. It does NOT support buying or selling of stocks.

---

## 🚀 Features

### 📈 Market Dashboard

* Overview of major indices like **NIFTY 50** and **Sensex**
* Displays trending stocks
* Top gainers & losers

### 🔍 Stock Search

* Search stocks listed in NSE/BSE
* Autocomplete suggestions for faster search

### 📊 Stock Details Page

* Current stock price
* Market Cap, PE Ratio, EPS
* Small-cap / Mid-cap / Large-cap classification
* Basic price visualization

### 🤖 AI-Based Stock Analysis

* Simple explanation of stock performance
* Strengths and weaknesses
* Risk level (Low / Medium / High)
* Recommendation: Buy / Hold / Avoid
* Time horizon: Short / Mid / Long term

### 📰 News & Sentiment Analysis

* Latest stock-related news
* Sentiment classification:

  * Positive
  * Negative
  * Neutral

### ⭐ Watchlist

* Add and remove stocks
* Data stored locally in browser

### ⚖️ Stock Comparison

* Compare multiple stocks side-by-side

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/sharesapp.git
cd sharesapp
```

---

### 2️⃣ Setup Frontend (Client)

```bash
cd client
npm install
npm run dev
```

---

### 3️⃣ Setup Backend (Server)

Open a new terminal:

```bash
cd server
npm install
node index.js
```

## 🌐 Running the Application
